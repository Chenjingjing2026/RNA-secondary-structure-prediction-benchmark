### Download Pre-trained Models

Our pre-trained models with BERT, ERNIE and MOTIF masking strategies can be downloaded from Google Drive (https://drive.google.com/drive/folders/1Ls5k7hv83BLRTznB4XcegIa2yKkU40Ls?usp=drive_link).

After downloading, please place the `.pdparams` and `.json` files into the `./output/BERT,ERNIE,MOTIF,PROMPT` folder. If this directory does not exist, please create it manually.

> For more details on model usage, downstream tasks, or the latest updates, please refer to the official repository: https://github.com/CatIIIIIIII/RNAErnie
